# Instructions
TO export your chats, follow these instructions
| Platform | Export Format | How to Export |
|----------|--------------|---------------|
| **WhatsApp** | `.txt` | Open chat → ⋮ → More → Export chat (without media), which will download a `.txt` file. |
| **iMessage** | `.pdf` | macOS Messages app → Select conversation, scroll up as far as you want → File → Print → Save as PDF |
| **Facebook Messenger** | `.json` | [Go here](https://accountscenter.facebook.com/info_and_permissions/dyi/) and select to export in `.json` format. In the extracted folder, go to `your_facebook_activity/messages/inbox/[person_name]` and download `messages_1.json`.
| **Signal** | `.json` | Not possible on newer phones sadly |
| **WeChat** | `.txt` | Tried out as well, and sadly does not work |
